# ∞Care v20.1 Daily Hub

毎日まめの気軽さで、あすけんに入れていた体重・体脂肪率・体温・食事内容もまとめて残すための v20.1 プロトタイプです。

## v20.1で変えたこと

- スマホ表示を見やすく調整
  - キャラ立ち絵をスマホでは小さい横カード化
  - 気分ボタンを大きめの2列表示に変更
  - チップ・入力欄・カード余白をスマホ向けに調整
  - 詳細入力は折りたたみ中心にして、最初の画面を軽くしました
- TOPにパスワードロックを追加
  - 初期パスワード：`1109`
  - 設定画面から変更できます
  - 「施錠」ボタンでその場でロックできます

## 使い方

1. zipを展開します。
2. `index.html` をブラウザで開きます。
3. パスワード `1109` を入れて開きます。
4. 「今日」画面で気分、感情、体調、食事、体重、睡眠などを入力して保存します。

データはブラウザの localStorage に保存されます。別端末へ移す時やバックアップしたい時は、「ログ」画面から JSON エクスポートしてください。

## 主な機能

- 1日1枚の `dailyEntries` 型に統合
- 気分、感情、体調、今日やったこと、外出、セルフケア、嗜好品、天気、体重、体脂肪率、体温、お通じ、食事内容、歩数、運動、睡眠、生理、一行日記、今日気に入ったものを記録
- カレンダーで日ごとの記録を表示
- グラフで気分・睡眠・タグ頻度を確認
- JSONエクスポート／インポート対応

## 公開するときの注意

このアプリ内パスワードは、公開URLを開いた人に中身を見せないための「簡易ロック」です。
静的サイトとしてGitHub Pagesなどに置く場合、アプリのコード自体は公開されるため、強いセキュリティにはなりません。

ただし、記録データは基本的に開いた端末のブラウザ内に保存されるため、他人が同じURLを開いても南帆の端末に保存された記録がそのまま見えるわけではありません。
本当に第三者に開かれたくない場合は、Cloudflare Access、Basic認証、Netlify/Vercel等のパスワード保護など、公開先側の認証も併用してください。
